Principais formatações

Texto em negrito: usar a tag strong.

Texto em itálico: usar a tag EM.

Texto marcado: usar a tag mark.

Texto grande e pequeno: usar as tags big e small.
Texto deletado: usar a tag del.
Texto com ênfase: usar a tag ins.
Texto com subscrição(dando um jeito melhor para explicações) como: H2O ou x20+3
Código-fonte(EXEMPLO: O comando document.get.ElementById('teste') é escrito em JavaScript): usar a tag code.
Tambem temos uma ótima formatação que e usada para explicar códigos.

Seria ela a tag pre. Exemplos: 
                
                    num = int(input('Digite um número'))
                    if num % 2 == 0:
                        printf(f'O número {num} é PAR')
                    else:
                        printf(f' O número {num} é ÍMPAR')
                    printf('Fim do programa')
Texto com citações simples: usar tag q.

Textos abreviados

HTML: abreviação de HTML usar abbr

CSS: abreviação de CSS sar tag abbr
